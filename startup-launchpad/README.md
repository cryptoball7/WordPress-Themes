Startup Launchpad
-----------------
A lightweight WordPress starter theme focused on SaaS/product landing pages.

Features:
- Video hero background with overlay
- Newsletter signup (AJAX) with shortcode [slp_newsletter]
- Product demo page template
- Responsive layout and basic accessibility considerations

Developer notes:
- Integrate a real email provider by editing slp_handle_subscribe() in functions.php (place API keys in wp-config.php or use a plugin).
- Consider adding a custom post type for 'demos' and Advanced Custom Fields (ACF) for richer content.
