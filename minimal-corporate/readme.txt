== Description ==
Minimal Corporate is a clean, typography-focused WordPress starter theme suitable for small businesses and consultants. It offers flexible homepage sections (as widget areas) and a simple shortcode-based pricing table.

== Installation ==
1. Zip the theme directory and upload via Appearance > Themes > Add New > Upload Theme or place in wp-content/themes/minimal-corporate.
2. Activate the theme.
3. Use Appearance > Widgets to populate "Homepage Section 1/2/3" widget areas or use the default fallback sections.
4. Customize hero text & accent color in Appearance > Customize.
